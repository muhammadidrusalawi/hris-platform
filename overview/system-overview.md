# System Overview

HRIS is a web-based Human Resource Information System
built with separated frontend and backend.

## Components
- Frontend: https://github.com/muhammadidrusalawi/hris-web (React.js)
- Backend: https://github.com/muhammadidrusalawi/hris-api (Laravel REST API)
- Database: PostgreSQL

## High-Level Flow
User → Frontend → Backend API → Database
